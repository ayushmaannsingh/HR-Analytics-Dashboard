📊 HR Analytics Dashboard | Power BI Project

This project is an interactive HR Analytics Dashboard developed in Microsoft Power BI to analyze employee data and uncover workforce insights. The dashboard focuses on employee attrition analysis, helping organizations understand trends related to employee turnover.

Key Features:
Employee Count & Attrition KPIs
Attrition Rate Analysis
Average Age, Salary & Years at Company
Attrition by Gender, Education & Age Group
Salary Slab Analysis
Job Role-wise Attrition Insights
Department-Level Filtering
Tools & Skills Used:
Microsoft Power BI
Data Cleaning & Transformation
DAX Measures
Data Visualization
Dashboard Design
Business Goal:

The dashboard helps HR teams identify patterns in employee attrition and supports better workforce decision-making through data-driven insights.
